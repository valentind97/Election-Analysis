# Election-Analysis
## Overview of Election Audit
In this Election Audit analysis we have been prvied a csv file containing election results form an election worker in Colorado. We will be able to deliver the following information by writing a code in python. 
- Total number of votes cast
- A complete list of candidates who received votes
- Total number of votes each candidate received
- Percentage of votes each candidate won
- The winner of the election based on popular vote


## Election Audit Results

Total votes: 369,711
### County votes:
- Jefferson: 10.5% (38,855 votes)
- Denver: 82.8% (306,055 votes)
- Araphoe: 6.7% (24,801 votes)

County with the largest turnout of voters was Denver 

### Candidate votes
- Charles Casper Stockham: 23.0% (85,213 votes)
- Diana DeGette: 73.8% (272,892 votes)
- Raymon Anthony Doane: 3.1% (11,606 votes)

From the results we can see that Diane Degette won with a total of 272,892 votes with a dominating percentage of 73.8%

## Election Audit Summary

From this script we can see that we have obtained the critical information out of this data file provided. Being able to quickly figure who won the election and how the vote spread was is very useful informatio. Then it was summed up to a nice and easy to understand text file. This script is not only limited to this election but also can be modified to work with any election. For example one thing that can be modified is to just change the csv file being read by the program. As well you can change the rows being read incase the other files have a different formatting.
